# 数据分析项目

这是一个数据分析工具的代码仓库，主要用于处理和可视化实验数据。

## 快速开始

```bash
pip install -r requirements.txt
python main.py --input data/analysis_data.json --config config.json
```

## 文件结构

- `data/` - 原始数据文件
- `config/` - 配置文件
- `notes/` - 项目笔记
